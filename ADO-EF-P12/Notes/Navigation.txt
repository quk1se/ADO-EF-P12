Навігаційні властивості
Це властивості (properties-{get;set;}) у класах-сутностях (Entity),
які відповідають за зв'язки з іншими сутностями.

public class Manager
{
    public Guid IdMainDep   { get; set; }    // властивість (даних)
    public Department MainDep { get; set; }  // навігаційна властивість
        
}
Це дозволяє спростити одержання зв'язаних даних - не подавати окремий 
запит на пошук, а скористатись неявними можливостями

Після оголошення навігаційних властивостей їх треба налаштувати
у контексті даних (див. клас DataContext)
Після налаштування реляцій треба зробити та застосувати міграцію
